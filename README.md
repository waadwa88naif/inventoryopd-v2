inventoryopd-v2

Google Apps Script web app for outpatient pharmacy medication availability.

Files:
- Code.gs
- PharmacistDashboard.html
- SupplyPanel.html
- index.html

Main source of truth:
- Master_DB = current medication state
- Update_Log = historical availability update log
- Expiry_Dates = expiry dates
- Settings = users and dropdown values
